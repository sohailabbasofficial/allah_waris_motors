1. Start / Login

Splash Screen

Kaam: App khulte hi thodi der branding dikhati hai.

  • App name: Allah Waris Motors
  • Tagline, Version, Developer credit
  • Koi button nahi – automatically agay chali jati hai

Owner Sign-In (Google)

Kaam: Workshop owner Google account se app unlock karti hai.

  • Sign in with Google – Google account choose karein
  • Sirf authorized email: allahwarismotors@gmail.com
  • Galt account → Access Denied → turant logout
Tip: Pehli baar sahi Google se login ke baad Fingerprint / Face Unlock chalu kar sakte hain (Settings mein).

Create PIN

Kaam: 4-digit PIN set karna (pehli baar / PIN reset ke baad).

  • Pehle PIN enter → phir Confirm PIN
  • Dono match hone chahiye

Login (Enter your PIN)

Kaam: PIN ya fingerprint se app kholna.

  • 4 digit PIN enter karein
  • Use fingerprint – agar on ho
  • Galat attempts zyada → thodi der lock

9. Backup, Restore & Sync

Backup Center

Kaam: Google Drive par database upload / manage.

  • Folder name: Car Rental Manager Backups
  • Backup Now – abhi upload (naya history file bhi ban sakta hai)
  • Internet + owner Google zaroori
  • Web browser mein Drive backup support nahi

Restore Backup

  • Drive se backup list → file select → Restore
  • Local data replace hoti hai (poori database)
  • Success ke baad Home

Recover Cloud Data (one tap)

Kaam: Latest Drive backup uthakar phone local data update.

Kahan mila: Empty Home banner · More → Recover Data · Settings · Restore screen

  1. Recover Cloud Data / Recover now
  2. Confirm
  3. Customers, ledger, payments load

Automatic sync (edits ke baad)

Customer / Transaction / Payment add, edit, delete ke ~4 seconds baad app quietly Drive par latest backup update karti hai (agar online + signed in).

Reinstall ke baad sahi tareeqa

  1. App install
  2. Owner Google Sign-In
  3. PIN set/enter
  4. Pehle Recover Cloud Data (Backup Now empty data se mat dabayein pehle)
  5. Phir normal use + auto sync / Backup Now
Important: Empty phone par pehle Backup Now mat chalao – pehle Recover/Restore karo. Agar Drive file Trash mein ho to pehle Google Drive Trash se Restore karein.

10. Important Tips (Asaan points)

  • Sirf allahwarismotors@gmail.com se app / backup chale ga.
  • Phone change / uninstall: pehle kabhi Backup / sync hona chahiye; naya phone → Recover.
  • “App is not safe” Play Protect message sideload APK ki wajah se hota hai – Install anyway choose karein (workshop apni APK).
  • Fingerprint Settings → Security mein, Google login ke baad.
  • Currency har jagah Rs format mein.
